# 🚀 C Basic Course Web

Một website học **Lập Trình C Cơ Bản** được xây dựng bằng **HTML + Vue.js + TailwindCSS**.

## 🌐 Demo

👉 https://minzdat.github.io/course-c-basic/

---

## 📌 Giới thiệu

Dự án này là một nền tảng học lập trình C trực quan, giúp người học:

* 📖 Học lý thuyết theo từng chương
* 💻 Xem ví dụ code thực tế
* ▶️ Chạy thử code (mô phỏng output)
* 🧪 Làm bài tập và xem lời giải
* 🌐 Chuyển đổi ngôn ngữ (VI / EN)

---

## 🧱 Công nghệ sử dụng

* HTML5
* CSS3 (TailwindCSS)
* JavaScript (Vanilla)
* Vue.js 3 (CDN)

## ⚙️ Cách chạy project

### 1. Clone project

```bash
git clone https://github.com/minzdat/course-c-basic.git
```

### 2. Mở web

Chỉ cần mở file:

```bash
index.html
```

👉 hoặc dùng Live Server trong VS Code

---

## 🚀 Deploy (GitHub Pages)

1. Vào Settings → Pages
2. Chọn:

   * Branch: main
   * Folder: /root
3. Save

👉 Website sẽ chạy tại:
https://minzdat.github.io/course-c-basic/
